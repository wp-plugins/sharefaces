ShareFaces is basically a Facebook share button that allows you to display users who have shared your article in the nice footer widget.

== Description ==

ShareFaces is a Facebook Share / Like Button that displays the Faces of Facebook users in a nice widget and specialized Footer Bar. There is nothing like actually using, so we recommend you to give it a shot!

== Installation ==

The installation is simple straight forward, you can use auto install directly from the admin panel or you can download the *.zip file and manually upload and activate it.

1. Download
1. Upload
1. Activate

Settings is quite self-explanatory, there is nothing so complicated about the plugin installation, but if you need any help you can visit our website and contact us.

== Frequently Asked Questions ==

= How do you use ShareFaces? =

It's easy! =] After activating ShareFaces, settings in the admin panel is optional. With one click installation, ShareFaces is the most simple and easy to use Share Button made available in the WordPress Plugin Directory.

= Why is it not working? =

There might be a conflict with some of the activated plugins, if you are using other Facebook like plugin, try deactivating them.
